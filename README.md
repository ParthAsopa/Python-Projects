# Lister - Your Discord List Management Assistant

Lister is a versatile Discord bot that helps you manage various lists such as to-do lists, watchlists, bucket lists, and more. Stay organized and boost your productivity with Lister!

## Features

- Create and manage different types of lists.
- Add tasks, mark them as completed, and remove them from your lists.
- Server-specific lists for enhanced privacy.

## Commands

- `!create <name>`: Create a new list.
- `!add <name> <task>`: Add a task to a list.
- `!complete <name> <task_number>`: Mark a task as completed.
- `!lists`: Display available lists.
- `!delete <name>`: Delete a list.
- `!clear <name>`: Clear all tasks from a list.
- `!show <name>`: Show tasks in a list.
- `!remove <name> <task_number>`: Remove a task from a list.
- `!help`: Display help message.
- `!save_lists`: Save the current state of lists.

## Usage

1. Invite Lister to your server: [Add Lister to your server](https://discord.com/api/oauth2/authorize?client_id=1204339568380485642&permissions=0&scope=bot).
2. Use the commands mentioned above to manage your lists.
