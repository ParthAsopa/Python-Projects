# Wi-Fi Firewall Login Automation

## Overview

This project is specifically designed to automate the login process for the Wi-Fi firewall of my hostel's Wi-Fi network. It streamlines the connection process, allowing users to access the internet quickly and efficiently without the need for manual login.

## Features

- Automated login to the Wi-Fi firewall
- Simplified user experience
- Fast and reliable connection setup

## How It Works

To use this project, simply run the provided batch file. This batch file activates the Python virtual environment and executes the main Python script. The login details for the Wi-Fi firewall are securely stored in a `.env` file, which is excluded from the repository to protect sensitive information.

## Disclaimer

This project is intended for personal use only and is specifically tailored for the Wi-Fi firewall of my hostel.
